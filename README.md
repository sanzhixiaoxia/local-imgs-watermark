# 本地图片加水印 (local-imgs-watermark)

一个完全在**本地浏览器**中运行的图片批量加水印工具，无需上传到任何服务器，保护隐私。支持文字水印、图片水印、平铺水印，并可一键批量导出。

## 功能特性

- **本地处理**：图片不上传服务器，全部在浏览器端用 Canvas 完成，隐私安全
- **多图批量处理**：可同时上传多张图片，批量添加水印并打包下载（基于 JSZip）
- **两种水印类型**
  - 文字水印：自定义内容、字号、颜色、透明度、旋转角度
  - 图片水印：上传一张 logo / 图片作为水印
- **9 种位置**：左上、上中、右上、左中、居中、右中、左下、下中、右下
- **平铺水印**：可开启平铺模式，铺满整张图片防止裁切去除
- **精细控制**：偏移量（X/Y）、透明度、旋转角度自由调节
- **粘贴网络图片**：直接 `Ctrl + V` 粘贴图片 URL（含微信公众号 `mmbiz.qpic.cn` 等 CDN 图片，已通过本地代理绕过跨域限制）
- **实时预览**：所见即所得，编辑参数即时更新预览效果
- **设置持久化**：水印配置自动保存在浏览器 `localStorage`
- **一键复制**：可将带水印的图片直接复制到剪贴板

## 快速开始

### 环境要求

- Node.js 18+
- npm（或 pnpm / yarn）

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

启动后浏览器访问终端提示的地址（默认 `http://localhost:5173/local-imgs-watermark/`）。

### 构建与预览

```bash
# 生产构建，输出到 dist/
npm run build

# 本地预览构建产物
npm run preview
```

## 使用说明

1. **添加图片**
   - 点击 / 拖拽上传本地图片
   - 或直接复制一张网络图片链接，在页面中 `Ctrl + V` 粘贴
2. **选择水印类型**
   - 文字：填写水印文字，设置字体大小、颜色、透明度、旋转角度
   - 图片：上传一张图片作为水印
3. **调整位置与样式**
   - 拖动位置按钮选择锚点
   - 开启「平铺」可让水印平铺满整图
   - 通过滑块微调偏移量、透明度和旋转
4. **预览与导出**
   - 左侧实时预览效果，多张图片可点击切换
   - 点击「批量下载」将所有带水印图片打包为 zip 下载
   - 或点击「复制图片」将当前预览图复制到剪贴板

## 关于微信公众号 / 跨域图片

微信公众号后台图片托管在 `mmbiz.qpic.cn` / `mmbiz.qlogo.cn` 等 CDN 上，这些域名**不返回 CORS 头**，浏览器直接加载到 Canvas 会被「污染」导致无法导出。

本项目在开发服务器中内置了一个图片代理中间件（`vite.config.ts` 中的 `image-cors-proxy`）：

- 粘贴受限域名的图片 URL 时，应用会先通过本地代理 `/<base>/api/image-proxy?url=...` 拉取图片
- 代理在服务端（无跨域限制）抓取原图，并带上 `Access-Control-Allow-Origin` 头回传
- 浏览器拿到的图片与站点同源，Canvas 不再被污染，可正常加水印并导出

> 说明：该代理基于 Vite 开发服务器（`npm run dev`），仅在本地开发模式下生效。若部署为静态站点（`npm run build` 产物），需自行在后端部署等价的图片代理服务。

## 技术栈

| 分类 | 技术 |
| --- | --- |
| 框架 | React 18 + TypeScript |
| 构建 | Vite 6 |
| 样式 | Tailwind CSS 3 |
| 打包导出 | JSZip + file-saver |
| 图标 | lucide-react |
| 核心 | Canvas 2D（水印绘制、预览、导出） |

```

├── components/
│   ├── Header.tsx     # 顶部栏（批量下载、复制、清空）
│   ├── ImageUpload.tsx# 图片上传/粘贴入口

│   └── WatermarkConfig.tsx # 水印参数配置面板
├── utils/

└── index.css          # 全局样式
vite.config.ts         # Vite 配置 + 图片代理中间件
```
